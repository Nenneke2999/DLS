**Цель проекта**: освоение ключевых концепций машинного обучения, нейронных сетей и их применение в реальных задачах, таких как обработка изображений, семантическая сегментация, детекция объектов и генеративные модели.

### Содержание проектов
- game_of_thrones.ipynb - Библиотека sklearn и классификация с помощью KNN
- kaggle_competition.ipynb - Применение линейных моделей для прогнозирования оттока клиентов
- kaggle_Simpsons.ipynb - Классификация изображений (Кастомная CNN на PyTorch, F1-оптимизация)
- semantic_segmentation.ipynb - Сегментация медицинских изображений (U-Net vs SegNet, Dice/Focal loss)
- autoencoders.ipynb - вариационные автоэнкодеры (Vanilla Autoencoder, Variational Autoencoder, Conditional VAE)
- WGAN_GP.ipynb - генеративно-состязательные модели (Wasserstein GAN with Gradient Penalty)
